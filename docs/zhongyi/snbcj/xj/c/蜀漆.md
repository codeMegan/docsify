## 蜀漆
<p>&emsp;&emsp;
味辛平。
</p>
<p>&emsp;&emsp;
主疟及咳逆，寒热，腹中症坚，痞结，积聚邪气，蛊毒，鬼注（旧作，《御览》作蛀）。生川谷。
</p>
<p>&emsp;&emsp;
《吴普》曰：蜀漆叶，一名恒山，神农岐伯雷公辛有毒，黄帝辛，一经酸，如漆叶蓝青相似，五月采（《御览》）。
</p>
<p>&emsp;&emsp;
《名医》曰：生江陵山，及蜀汉中常山，苗也，五月采叶，阴干。
</p>
<p>&emsp;&emsp;
案《广雅》云：恒山蜀漆也。《范子计然》云：蜀漆出蜀郡。
</p>
